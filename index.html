Abstract

Text-based video editing aims to generate videos that align with given text descriptions and maintain faithfulness with source videos. Prior works have made effective progress in terms of temporal consistency. However, they usually require an overdetailed description of the input video's content, which can be cumbersome. At the same time, prior works mainly focus on overall visual editing, ignoring we sometimes target local area editing but face surroundings inconsistency with the input. In this paper, we propose semantic video editing to simplify overdetailed descriptions for editing. Furthermore, we propose a region controllable Semantic video Editing framework to address surroundings inconsistency and improve source faithfulness. SemEdit automatically generates semantic masks via clustering intermediate features, emphasizing regions that need to be edited. Moreover, we conduct extensive experiments on the DAVIS dataset and various types of text to demonstrate the effectiveness of the proposed framework.

Method

We propose a method to equip pre-trained Text-to-Video (T2V) models with a style adapter, allowing for the generation of stylized videos based on both a text prompt and a style reference image. The overview diagram is illustrated as the following figure. In this framework, the textual description dictates the video content, while the style image governs the visual style, ensuring a disentangled control over the video generation process.

Given the limited availability of stylized videos, we employ a two-stage training strategy. Initially, we utilize an image dataset abundant in artistic styles to learn reference-based style modulation. Subsequently, adaptation finetuning on a mixed dataset of style images and realistic videos is conducted to improve the temporal quality of the generated videos.
